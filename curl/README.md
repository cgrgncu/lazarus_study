# 使用外部curl程式的範例

### 下載curl.exe
+ 官網: https://curl.se/
+ 官方整理各種下載頁面: https://curl.se/download.html
+ 比較推薦用「Muldersoft」版本，屬於超極致單一獨立執行檔，沒有DLL檔案:
  + 主要下載庫: https://sourceforge.net/projects/muldersoft/files/cURL/
  + 選擇版本: curl-8.20.0-win-x86-full.2026-05-02.zip
  + 裡面有用的只有EXE和crt。
  + CRT效期通常只有一年，沒事就不要用。
  + 可以透過--ssl-revoke-best-effort改用微軟內建的
