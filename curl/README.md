# 使用外部curl程式的範例

### 下載curl.exe
+ 官網: https://curl.se/
+ 下載頁面: https://curl.se/download.html
+ 官方最新windows版下載頁面: https://curl.se/windows/
+ 選用: curl-8.20.0-win-x86-full.2026-05-02.zip
+ https://sourceforge.net/projects/muldersoft/files/cURL/
+ 要選定指定版本下載，要自己按照規則弄連結:
  + 這裡選擇:
    + curl version: 8.20.0
    + Build: 8.20.0_5
    + Date: 2026-04-29
  + 其下載連結: https://curl.se/windows/dl-8.20.0_5/curl-8.20.0_5-win64-mingw.zip
